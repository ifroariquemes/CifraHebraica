## Como usar

```
$ hebrew -h

---- Instituto Federal de Rondônia - Campus Ariquemes ----
Criptografa e descriptografa mensagens usando Cifra Hebraica (ATBASH, ATBAH ou ALBAM)
N. A. V. Simões <natanael.simoes@ifro.edu.br>
http://github.com/ifroariquemes/CifraHebraica

Uso:

      hebrew -c|-d mensagem [-m metodo]

 -c mensagem      Criptografa uma mensagem
 -d mensagem      Decriptografa uma mensagem
 -m metodo        Estabelece o metodo atbash(padrao), atbah ou albam
 -h               Mostra essa mensagem

 Criptografar: hebrew -c mensagem
 Decriptografar: hebrew -d NVMHZTVN
 ```
